Performance Task for Lesson 6

Posted on January 7, 2021

Create a program that asks user to input integer repeatedly, and it stop and display output once input a negative number.

=====WHILE LOOP EXERCISE=====
Enter a number: 1
Enter a number: 2
Enter a number: 3
Enter a number: 4
Enter a number: 5
Enter a number: 6
Enter a number: -7
No. of Odd numbers: 4
No. of Even numbers: 3
Total No. of Attempts: 7

int ii = 0, inp, odd = 0, even = 0;
List<int> num = new List<int>();

Console.WriteLine("=====WHILE LOOP EXERCISE=====");

while (true) {
 Console.Write("Enter a number: ");
 inp = Convert.ToInt32(Console.ReadLine());
 num.Add(inp);

 if (inp % 2 == 0)
  even++;
 else
  odd++;

 if (!Convert.ToString(inp).Contains("-"))
  continue;
 else
  break;
}

Console.WriteLine("No. of Odd numbers: {0}", odd);
Console.WriteLine("No. of Even numbers: {0}", even);
Console.WriteLine("Total No. of Attempts: {0}", num.Count);
Console.ReadLine();

Using nested loop to print the following pattern.

*4321
5*321
54*21
543*1
5432*

int i, j;
for (i = 5; i >= 1; i--) {
 for (j = 5; j >= 1; j--) {
  if (j != i)
   Console.Write(j);
  else
   Console.Write("*");
 }
}
Console.ReadLine();

Using do-while loop to print the following output.

1 little 2 little 3 little indian
4 little 5 little 6 little indian
7 little 8 little 9 little indian
10 little indian boys

int i = 1;
do {
 if (i == 10)
  Console.WriteLine("10 little indian boys");
 else if (i % 3 == 0)
  Console.WriteLine("{0} little indian", i);
 else
  Console.Write("{0} little ", i);
 i++;
} while (i <= 10);
Console.ReadLine();
