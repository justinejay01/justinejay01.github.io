Procedure & Function Exercise

Exercise: Take user input for arithmetic operation, and create GetAnswer method. Method must have 3 parameters.

Enter number: 11
Enter number: 26
Enter operator (+, -, *, /): +

11 + 26 = 37
Procedure

In programming, procedure is a set of instructions which takes input and performs certain task, but does not return a value. In C#, it uses void method.

public static void Main(string[] args)
{
 int x, y;
 char z;
 Console.Write("Enter number: ");
 x = Convert.ToInt32(Console.ReadLine());
 Console.Write("Enter number: ");
 y = Convert.ToInt32(Console.ReadLine());
 Console.Write("Enter operator (+, -, *, /): ");
 z = Convert.ToChar(Console.ReadLine());
 Console.WriteLine();
 GetAnswer(x, y, z);
 Console.ReadLine();
}

public static void GetAnswer(int x, int y, char z)
{
 switch (z)
 {
  case '+':
   Console.WriteLine("{0} {1} {2} = {3}", x, z, y, x + y);
   break;
  case '-':
   Console.WriteLine("{0} {1} {2} = {3}", x, z, y, x - y);
   break;
  case '*':
   Console.WriteLine("{0} {1} {2} = {3}", x, z, y, x * y);
   break;
  case '/':
   Console.WriteLine("{0} {1} {2} = {3}", x, z, y, x / y);
   break;
  default:
   Console.WriteLine("Invalid input!");
   break;
 }
}
Function

In programming, function is also a set of instructions which takes input, performs certain task, and returns a value. In C#, it uses any value-returning functions.

public static void Main(string[] args)
{
 char w;
 int x, y, z;
 Console.Write("Enter number: ");
 x = Convert.ToInt32(Console.ReadLine());
 Console.Write("Enter number: ");
 y = Convert.ToInt32(Console.ReadLine());
 Console.Write("Enter operator (+, -, *, /): ");
 w = Convert.ToChar(Console.ReadLine());
 z = GetAnswer(w, x, y);
 Console.WriteLine();
 
 if (w == '+' || w == '-' || w == '*' || w == '/')
  Console.WriteLine("{0} {1} {2} = {3}", x, w, y, z);
 else
  Console.WriteLine("Invalid input!");
 
 Console.ReadLine();
}

public static int GetAnswer(char w, int x, int y)
{
 int z = 0;
 switch (w)
 {
  case '+':
   z = x + y;
   break;
  case '-':
   z = x - y;
   break;
  case '*':
   z = x * y;
   break;
  case '/':
   z = x / y;
   break;
  default:
   break;
 }
 return z;
}
